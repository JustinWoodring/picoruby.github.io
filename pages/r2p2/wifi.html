This is a web-based tool to configure WiFi settings of R2P2 devices via Bluetooth Low Energy (BLE).

Show Password
ex: "JP" for Japan, "SG" for Singapore, "UK" for United Kingdom
If you choose No, manual reboot is required when WiFi is disconnected.

How to Use

  1. Start the device in WiFi configuration mode. The method varies by device:
    • For Raspberry Pi Pico W and Pico2 W: Boot the device with pin 22 connected to GND.
    • For other microcontrollers: Refer to the device manual.
    • You can also run the nmble command in the R2P2 shell.
  2. Ensure Bluetooth is enabled on your PC or Android device.
  3. Click the "Connect to Device" button to scan for available BLE devices.
  4. Look for "R2P2 BLE" in the list. If it doesn't appear, the device may not be functioning correctly.
    • The name may differ based on the production. See the manual.
  5. Pair with "R2P2 BLE" to display the configuration form.
  6. Fill in all the fields in the form.
  7. Click the "Send Configuration" button to submit the WiFi settings to the device.
  8. Once successful, reboot the device to connect it to the WiFi network. (Make sure the pin 22 is open)

Supported OSes and Browsers

Windows ✅ Chrome and Edge
Linux ✅ Chrome
macOS ✅ Chrome
Android ✅ Chrome and Edge
iOS ❌ No BLE allowed in Web Browsers in iOS

Report an issue if any correction is needed.

If the BLE doesn't work

You can configure by nmcli command in R2P2 shell.
